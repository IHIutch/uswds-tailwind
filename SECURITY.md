# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any `@uswds-tailwind/*` package, please report it privately so we can address it before public disclosure.

**Do not open a public issue for security problems.**

To report a vulnerability, please [privately report it via the Security tab](https://github.com/IHIutch/uswds-tailwind/security/advisories/new).

You can also email the maintainer directly at the email listed in this repository's [GitHub profile](https://github.com/IHIutch).

All security vulnerabilities will be promptly verified and addressed.
